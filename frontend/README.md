# Expense Tracker - DevOps Project

## Description
A simple full-stack application demonstrating DevOps pipeline using:
- Node.js (Backend)
- HTML/JS (Frontend)
- GitHub, Jenkins, Docker, Kubernetes, AWS, Nagios

## Features
- Add expenses
- View expenses

## Architecture
Frontend → Backend → CI/CD Pipeline → Deployment

## DevOps Flow
Developer → GitHub → Jenkins → Docker → Kubernetes → AWS → Nagios
